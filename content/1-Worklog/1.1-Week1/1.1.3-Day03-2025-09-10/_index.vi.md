# **Ghi chú Bài học**

## Bộ công cụ quản trị AWS

### AWS Management Console

- Đăng nhập bằng Root User hoặc IAM User (cần ID tài khoản 12 chữ số).  
- Tìm kiếm và mở dashboard của từng dịch vụ.  
- Support Center cho phép tạo ticket hỗ trợ trực tiếp.

![image](/images/1-Worklog/Week1/Screenshot_2025-09-11_132420.png)

### AWS Command Line Interface (CLI)

- Công cụ dòng lệnh mã nguồn mở tương tác với dịch vụ AWS.  
- Cung cấp tính năng tương đương Console.

![image](/images/1-Worklog/Week1/image.png)

**Đặc điểm chính:**

- Hỗ trợ đa nền tảng (Windows, macOS, Linux).  
- Dễ script và tự động hóa.  
- Truy cập trực tiếp API dịch vụ AWS.  
- Quản lý nhiều tài khoản thông qua profiles.

### AWS SDK (Software Development Kit)

- Đơn giản hóa việc tích hợp dịch vụ AWS vào ứng dụng.  
- Tự động xử lý xác thực, retry và tuần tự hóa dữ liệu.

![image](/images/1-Worklog/Week1/image%201.png)

**Ngôn ngữ hỗ trợ:**

- Python (Boto3)  
- JavaScript/Node.js  
- Java  
- .NET  
- Ruby, PHP, Go và các ngôn ngữ khác
