## Tổng quan tuần 4

### Nội dung chính

- Amazon S3 và các lớp lưu trữ.  
- S3 Static Website Hosting.  
- S3 Glacier cho lưu trữ dài hạn.  
- AWS Snow Family.  
- AWS Storage Gateway.  
- Chiến lược Disaster Recovery.  
- AWS Backup.
