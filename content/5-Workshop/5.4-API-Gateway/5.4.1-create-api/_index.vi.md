## Tạo API Gateway (REST)

1) Vào API Gateway → **Create API** → **REST API** → Build.  
2) Name: `serverless-workshop-api`; Endpoint type: Regional → Create API.  
3) Trong Resources: chọn `/` → **Create Resource** → Path: `hello` → Create.  
4) (Tùy chọn) bật CORS cho resource nếu muốn gọi từ browser.
